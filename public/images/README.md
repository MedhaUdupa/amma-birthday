# Images Directory

You can add as many images as you want! The website will automatically detect and display all of them.

## Image Naming

Name your images sequentially:
- `memory1.jpeg` - First memory image
- `memory2.jpeg` - Second memory image  
- `memory3.jpeg` - Third memory image
- `memory4.jpeg` - Fourth memory image
- `memory5.jpeg` - Fifth memory image
- ... and so on (up to 20 images)

## Image Requirements

- Format: JPEG
- Recommended size: 800x600px or larger (portrait orientation works best)
- The images will be automatically resized and optimized for the website

## Adding Images

1. Copy your images to this folder (`public/images/`)
2. Rename them to `memory1.jpeg`, `memory2.jpeg`, `memory3.jpeg`, etc. (sequential numbering)
3. The website will automatically detect and display all images in the carousel
4. No need to update any code - just add more images with the next number!

## Example

If you have 6 images:
- memory1.jpeg
- memory2.jpeg
- memory3.jpeg
- memory4.jpeg
- memory5.jpeg
- memory6.jpeg

All 6 will automatically appear in the carousel!
