# Images Directory

Please add your 4 images here with the following names:

- `memory1.jpeg` - First memory image
- `memory2.jpeg` - Second memory image  
- `memory3.jpeg` - Third memory image
- `memory4.jpeg` - Fourth memory image

## Image Requirements

- Format: JPEG
- Recommended size: 800x600px or larger
- The images will be automatically resized and optimized for the website

## Adding Images

1. Copy your images to this folder (`public/images/`)
2. Rename them to `memory1.jpeg`, `memory2.jpeg`, etc.
3. The website will automatically display them in the Memories section
